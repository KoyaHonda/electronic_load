## 電子負荷装置にシーケンスを書き込む
- ２つの方法について機能と制約を調べる
    - シーケンスをあらかじめ書き込む
        - シーケンス機能
            - 電流値,電圧値,電力値,コンダクタンス値が設定可能
            - ステップ実行時間 : 25 μs ~ 3.6 Ms
            - 編集,実行,停止はリモート操作可能
            - 合計30個のプログラムを設定可能
            - プログラムのループ回数 : 1回 ~ 10万回
        - 書き込み方法
            - 「電子負荷装置 PLZ^5Wシリーズ 通信インターフェースマニュアル」参照
                チュートリアル > シーケンス動作

    - PCからリアルタイムに送る
        - 時間管理が面倒
        
- コマンドの基本
    - コマンドは「電子負荷装置 PLZ^5Wシリーズ 通信インターフェースマニュアル」参照
    - SCPIコマンドの小文字は省くことができる
    - コマンド入力の際大文字、小文字の区別はない
    - { } 内の ” | ” で区切られた文字は、その中から一つを選択
    - < > 内の文字はパラメータ
    - [ ] 内の文字はオプションデータ、省略した場合はデフォルトの値が入る
    - { }, < >, [ ] は実際のコマンドには記入しない]

- プログラム上の基本コード
    - my_serial.open_port()    # COMポートを開く
    - my_serial.close_port()   # COMポートを閉じる
    - my_serial.query(query)   # queryを送信
    - my_serial.send(command)  # comanndを送信


## 電子負荷装置フロントパネル操作
- 測定値の記録
    1. MEASURE > Acquire キーを押す
    2. ロータリノブでトリガの設定を行う
    2. Initiate キーを押して測定開始
    3. Abort キーを押して測定停止

- シーケンス実行方法
    1. 動作モードとレンジを実行方法に合わせて設定する
    2. SEQENCE > PROGRAM キーを押す
    3. ロータリノブで実行したいプログラムを選択し、SELECT キーを押す
    4. Transient > Initiate キーを押す
    
- シーケンス削除方法
    1. SEQENCE > PROGRAM キーを押す
    2. ロータリノブで削除したいプログラムを選択し、Delete
    3. ロータリノブでOKを選択し、ENTER キーを押す
